# Grid

## Containers

Containers provide a means to center and horizontally pad your siteâ€™s contents. Use Container for a responsive pixel
width or FluidContainer for width: 100% across all viewport and device sizes.

### Container
Two different types of containers, the main container and fluid container. The primary difference being the fluid
container is always 100% width, whereas the main container has a max-width of 1200px. The main container also has auto
margins to keep centred.

The grid also uses a 12 column based system, which is built on top of this [library](https://github.com/LoicMahieu/react-styled-flexboxgrid).
