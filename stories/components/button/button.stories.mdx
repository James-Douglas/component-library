# Button

The Button component has four available variants: `hero`, `primary`, `secondary`, `tertiary`.

The Button can be styled with an icon and can be configured according to the design system.

All Button variants bar `tertiary` have an optional `inverted` prop for use against dark backgrounds.

Button width is set to `100%` by default, so the Button fills the width of its parent container. The width
attribute can be overridden via the `width` prop if desired.

As an escape hatch for situational styling, the style prop can be used for cases where the button needs to be aligned in a different flex position, for example.

The tagtype prop can be used to set the html type attribute, for example setting the button to be a `submit` type.

If the `href` prop is supplied, the underlying HTML tag will be an `a`, otherwise this will be a `button` tag. This does not
affect the styling, which is determined via the supplied `variant`.

### Props

<Props of={Button} />

<Story name="button with props" decorators={[withKnobs]} parameters={{ docs: { disable: true } }}>
  {() => {
    const id = text('id', 'btn1');
    const variant = select('variant', ['hero', 'primary', 'secondary', 'tertiary'], 'hero');
    const inverted = boolean('inverted', false);
    const children = text('content', 'Compare the market');
    const disabled = boolean('disabled',  false);
    const iconAlign = select('icon align', ['left', 'right'], 'left');
    const href = text('href', '#');
    const target = text('target', '_blank');
    const rel = text('rel', '');
    const iconButton = boolean('iconButton', false);
    return (
      <div style={{ padding: '2%', background: 'white' }}>
        <div>
          <Button
            trackingLabel="example button"
            id={id}
            variant={variant}
            inverted={inverted}
            disabled={disabled}
            icon={faInfoCircle}
            iconAlign={iconAlign}
            href={href}
            target={target}
            rel={rel}
            iconButton={iconButton}
          >
            {children}
          </Button>
        </div>
      </div>
    );
  }}
</Story>
