# Header

Manor offers both a Header and a StickyHeader. Header should be used when a Progress component (`ProgressBar`, `ProgressTracker`)
is used, as the Progress has sticky functionality. If there is no Progress component, the StickyHeader should be used.

The examples are best view in canvas mode.
