# FeatureList

The FeatureList component can be used to render a simple list of features/benefits.

<Preview>
  <Story name="feature list">
    <div style={{ padding: '2%'}}>
      <FeatureList features={['Visa Debit', 'Paywave', 'Apple Pay']} />
    </div>
  </Story>
</Preview>
