# Toggles

There are three Toggle components, documented below. All Toggles are typically
wrapped inside a ToggleGroup which manages labels, validation, tooltips, and
selection events.

## Toggle

The Toggle component is the primary toggle and is used to render square,
rectangle, icon and picture toggles:

#### Custom rectangle toggles

ToggleGroup can be passed a `rectOptions` object to customize the rectangle toggle.

        col: 4,
        height: 12,
        align: 'center',

      return (
        <Container>
          <ToggleGroup
            label="Rectangle with columns & height"
            name="toggleGroupE"

## CustomToggle

A CustomToggle allows you to define the contents of the toggle yourself.

<Preview>
  <Story name="custom toggle demo" >
    {() => {
      const StyledPictureContainer = styled.div`
        height: 14.2rem;
        width: 21.9rem;
      `;
      const StyledOtherContainer = styled.div`
        padding-top: 2rem;
        height: 12rem;
        width: 21.9rem;
        display: flex;
        justify-content: center;
        align-items: center;
        opacity: 0.2;
      `;
      const StyledLabelContainer = styled.div`
        display: flex;
        flex-direction: column;
        align-items: center;
        justify-content: center;
        height: 6rem;
      `;
      const StyledMedicareToggle = styled.div`
        height: 19.5rem;
        justify-content: space-between;
        display: flex;
        flex-direction: column;
        align-items: center;
      `;
      const tooltip = {
        title: 'test',
      };
      return (
         <Container>
           <ToggleGroup label="Medicare toggles" name="toggleGroupA" tooltip={tooltip} handleToggle={(value) => console.log(`TOGGLE SELECTED: ${value} `)}>
             <CustomToggle
               name="image"
               id="green"
               value="green"
             >
               <div className="medicare-toggle">
                 <StyledPictureContainer>
                   <Picture
                     src={DEFAULT}
                     srcsets={[{ srcset: DEFAULT }]}

                 </StyledPictureContainer>
                 <StyledLabelContainer>
                   <strong>Green</strong>
                 </StyledLabelContainer>
               </div>
             </CustomToggle>
             <CustomToggle
               name="image"
               id="blue"

                   <Picture
