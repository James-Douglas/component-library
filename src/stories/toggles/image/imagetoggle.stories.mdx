# ImageToggle

The ImageToggle component can be used to display a small image within a toggle. The image supplied will be sized to fit an
8rem square (80x80 pixels). The ImageToggle uses the Picture component to render the image, so further details on related
props can be found within the <LinkTo kind="components-picture--picture">Picture components documentation</LinkTo> (`src`, `srcsets`, `alt`, `pictureTitle`).

As with all Toggle components, ColorToggles should typically be nested within a <LinkTo kind="components-toggles-togglegroup--togglegroup">ToggleGroup</LinkTo> which manages the fields labels,
validation, tooltips and selection events.
