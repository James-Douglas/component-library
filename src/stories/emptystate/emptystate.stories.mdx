# EmptyState

The `EmptyState` component is to be used when no results are found based on the current filters.

<Preview>
  <Story name="empty state" height="500px">
    {() => {
      return (
        <div style={{marginTop: '32px'}}>
          <EmptyState>
            <div style={{fontSize: '20px', lineHeight: '30px'}}>
              <p>Try <a href="#" title="link">adjusting your search or filter</a> to find what you are looking for.</p>
            </div>
          </EmptyState>
        </div>
      );
    }}
  </Story>
</Preview>
