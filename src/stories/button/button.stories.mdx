# Button

The Button component has six available variants: `primary`, `secondary`, `tertiary`, `text`, `link` and `footer-link`.

The Button can be styled with an icon and can be configurable according to the design system.

All Button variants bar `primary` have an optional `darkMode` prop for use against dark backgrounds.

Button width is set to `100%` by default, so the Button fills the width of its parent container. The Button width
attribute can be overridden via the `width` prop if desired.

If the `href` prop is supplied, the underlying HTML tag will be an `a`, otherwise this will be a `button` tag. This does not
affect the styling, which is determined via the supplied `variant`.

### Props

<Props of={Button} />

<Story name="button with props" decorators={[withKnobs]} parameters={{ docs: { disable: true } }}>
  {() => {
    const id = text('id', 'btn1');
    const variant = select('variant', ['primary', 'secondary', 'text', 'link', 'footer-link'], 'primary');
    const darkMode = boolean('darkMode', false);
    const size = select('size', ['sm', 'md', 'lg'], 'md');
    const children = text('content', 'Compare the market');
    const disabled = boolean('disabled',  false);
    const iconSize = select('Icon size', ["lg","xs","sm","1x","2x","3x","4x","5x","6x","7x","8x","9x","10x"], '1x');
    const iconAlignRight = boolean('iconAlignRight', false);
    const href = text('href', '#');
    const target = text('target', '_blank');
    const rel = text('rel', '');
    return (
      <Button
        id={id}
        variant={variant}
        darkMode={darkMode}
        size={size}
        disabled={disabled}
        icon={faInfoCircle}
        iconSize={iconSize}
        iconAlignRight={iconAlignRight}
        href={href}
        target={target}
        rel={rel}
      >
        {children}
      </Button>
    );
  }}
</Story>
