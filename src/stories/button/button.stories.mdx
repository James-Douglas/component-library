# Button

The button can be styled with an icon, configurable for design approved modes. These are: `primary`, `secondary`,
`text`, `link` and `footer-link`. `secondary`, `text` and `link`. All have an `darkMode` mode for dark backgrounds apart
from `primary`.

When `text`, `link` and `footer-link` are supplied as a `variant`, the html structure with be rendered as an `<a>` element,
with removed padding/button styling.

### Props

<Props of={Button} />

<Story name="button with props" decorators={[withKnobs]} parameters={{ docs: { disable: true } }}>
  {() => {
    const id = text('id', 'btn1');
    const variant = select('variant', ['primary', 'secondary', 'text', 'link', 'footer-link'], 'primary');
    const darkMode = boolean('darkMode', false);
    const size = select('size', ['sm', 'md', 'lg'], 'md');
    const content = text('content', 'compare the market');
    const disabled = boolean('disabled',  false);
    const icon = select('icon', ['', 'info', 'contact', 'close'], '');
    const iconSize = number('iconSize', 2);
    const iconAlignRight = boolean('iconAlignRight', false);
    const href = text('href', '#');
    const target = text('target', '_blank');
    const rel = text('rel', '');
    return (
      <Button
        id={id}
        variant={variant}
        darkMode={darkMode}
        size={size}
        content={content}
        disabled={disabled}
        icon={icon}
        iconSize={iconSize}
        iconAlignRight={iconAlignRight}
        href={href}
        target={target}
        rel={rel}
      />
    );
  }}
</Story>
