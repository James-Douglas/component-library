# CTM Typography Guidelines

Typography styles can be applied by

* Applying the specific class to an individual element
* Apply .manor-rich-text a container element, which will apply styles to tags (e.g. h1 will have appropriate styling),
  however this will not extend to classes such as `manor-body2` or where there isn't a matching HTML tag (e.g. microcopy).

### .manor-li
Use case: List Items - adds padding after each item

### .manor-li-last-item
Use case: Last child List Items - removes padding from last list item

### .manor-nested-list
Use case: nested lists - removes margin from bottom of nested list

<Preview>
  <Story name="nested list">
    <ul className="manor-ul">
