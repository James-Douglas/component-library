# CTM Typography Guidelines

Typography styles 

* The manor global component (global styled-component) handles all html tag styling. For example, h1-h6, p, ul li etc. Import at the top of your app tree as `<Manor/>`
* For non html elements/styling, such as `.subtitle` or `.microcopy`, add the class to the element. This is also provided by the manor global component.
