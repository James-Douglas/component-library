# Icon component
Basic icon component, uses svg tags and paths

## Usage

~~~js
<ManorIcon 
  name
  flipH
  flipV
/>
~~~

## Props

1. `name` *(required) string*
    * The name of the icon to be displayed.
2. `flipH` *(optional) boolean*
	  * Flip the icon horizontally.
3. `flipV` *(optional) string*
	  * Flip the icon vertically
